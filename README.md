## How to install required packages

To reinstall needed packages, in MOSIX3, go to:

~/Rlibrary/src (i.e. /homes/Rlibrary/src), and then

R CMD INSTALL ergm_3.1-0.tar.gz  
R CMD INSTALL tergm_3.1.1.tar.gz ....
and so on...
